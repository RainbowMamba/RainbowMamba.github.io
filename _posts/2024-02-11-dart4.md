# Operators  

## Arithmetic operators
Very simple.
```dart
int a = 3;
int b = 2;

print(a + b); // 5
print(a - b); // 1
print(a * b); // 6
print(a / b); // 1.5 (as a double)
print(a ~/ b); // 1 (as an integer)
print(a % b); // 1 (Remainder)
```

## Increment and Decrement
Prefix
```dart
int a = 10;
print(a++); // 10
print(a); // 11
print(a--); // 11
print(a); // 10
```
Postfix
```dart
int a = 10;
print(++a); // 11
print(a); // 11
print(--a); // 10
print(a); // 10
```

## Equality and Relational Operators
```dart
int a = 3;
int b = 4;
print(a == b); // false
print(a != b); // true
print(a > b); // false
print(a < b); // true
print(a >= b); // false
print(a >= b); // false
```
