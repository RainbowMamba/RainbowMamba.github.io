# Operators  

## Arithmetic operators
Very simple.
```dart
int a = 3;
int b = 2;

print(a + b); // 5
print(a - b); // 1
print(a * b); // 6
print(a / b); // 1.5 (as a double)
print(a ~/ b); // 1 (as an integer)
print(a % b); // 1 (Remainder)
```
